Week 3: Combinational Logic

Exploring the building blocks of computation, from basic arithmetic to data selection and decoding.

Learning Objectives

By the end of this week, you will be able to:

  • Design and analyze half-adders, full-adders, and ripple-carry adders.
  • Understand the function and application of decoders in memory addressing and data routing.
  • Differentiate between various types of decoders (e.g., 2-to-4, 3-to-8).
  • Apply combinational logic principles to build essential arithmetic and decoding circuits.
