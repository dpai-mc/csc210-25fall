Lecture Assignment: Combinational Circuit Analysis and Design

Total Points: 100 points

Due Date: End of Week 3

Part A: Circuit Analysis (40 points)

  1. 4-bit Ripple Carry Adder Analysis (15 points)
    • Determine sum and carry outputs for specific input combinations
    • Calculate worst-case propagation delay
    • Identify critical path and timing considerations
  2. 8:1 Multiplexer Function Implementation (15 points)
    • Analyze MUX circuit implementing 3-variable Boolean function
    • Derive truth table from circuit diagram
    • Write simplified Boolean expression using K-maps
  3. Combined Decoder/Encoder System (10 points)
    • Trace signals through 3:8 decoder connected to 8:3 priority encoder
    • Explain system function and identify issues
    • Suggest improvements for reliability and performance

Part B: Design Problems (45 points)

  1. Multi-Level Arithmetic Circuit (20 points)
    • Design circuit computing (A + B) √ó C for 3-bit numbers
    • Use only half-adders, full-adders, and AND gates
    • Provide complete schematic with signal labeling
    • Estimate gate count and propagation delay
  2. Data Selector with Enable (15 points)
    • Design 4-input data selector with 2 select lines
    • Include active-low enable functionality
    • Show implementation using basic gates only
    • Create truth table and timing diagram
  3. BCD-to-Decimal Decoder (10 points)
    • Convert 4-bit BCD input to 10 output lines
    • Include invalid BCD code detection
    • Show cascading method for larger BCD numbers

Part C: Application Analysis (15 points)

  1. Computer System Integration (15 points)
    • Explain multiplexer use in CPU register files
    • Describe decoder role in memory address decoding
    • Analyze trade-offs between decoder size and memory organization
    • Connect concepts to Von Neumann architecture principles
