The Language of Computers

Ever wondered how computers, which only understand "on" and "off", handle complex tasks? It's all thanks to different number systems. This guide will walk you through Decimal (the one we know), Binary (the computer's language), and Hexadecimal (the programmer's shorthand).

Part 1: Back to Basics - Decimal (Base-10)

Everything starts with what we already know. The decimal system uses ten digits (0-9). The position of each digit determines its value (place value). Let's break it down interactively.

Part 2: Speaking Computer - Binary (Base-2)

Binary is the computer's native language, using only two digits: 0 (off) and 1 (on). Each position represents a power of 2. Click or use the space/enter key on the "bulbs" below to turn bits on or off and see how they add up!

Part 3: The Programmer's Shorthand - Hexadecimal (Base-16)

Long binary strings are hard for humans to read. Hexadecimal (Base-16) solves this by using 16 digits: 0-9 and A-F. It's a compact way to represent binary data.

The Magic Bridge

The key is that **one hex digit represents exactly four binary digits** (a "nibble"). This makes conversion between them incredibly fast.

For example, the binary string 11010110 can be split into 1101 and 0110. This translates to D and 6, giving you D6 in hex.

Part 4: Real-World Uses

These systems aren't just academic; they're everywhere in the digital world. Here are a few common examples where you'll see hexadecimal in action.

Web Colors

Hex codes are used to define colors in web design. The format is #RRGGBB, where each pair represents the intensity of Red, Green, and Blue.

#748E63

MAC Addresses

Every network-connected device has a unique Media Access Control (MAC) address, written as six pairs of hexadecimal digits, like 00:1A:2B:3C:4D:5E.

Error Codes

Many system error codes, like the infamous Windows "Blue Screen of Death" stop codes, are represented in hexadecimal, such as 0x0000007B.
