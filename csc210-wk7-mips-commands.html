🎯 Three Main Instruction Types

🔀 Branch Instructions

🎯 Complete Branch Reference

🔢 Two-Register Branches

beq rs, rt, label
Branch if Equal
000100
Branches if rs == rt
bne rs, rt, label
Branch if Not Equal
000101
Branches if rs != rt

🎯 Single Register Branches

Branch if Less Than Zero
000001
Branches if rs < 0
bgez rs, label
Branch Greater or Equal Zero
000001
Branches if rs ≥ 0

🚀 Jump Instructions

📚 Complete Jump Reference

🌐 J-Type Jumps

j target
Jump
000010
Unconditionally jump to target
PC = (PC+4)[31:28] | address << 2
jal target
Jump and Link
000011
Jump and save return address in $ra
$ra = PC + 4, then jump

📍 R-Type Jumps

jr rs
Jump Register
001000
Jump to address in register rs
PC = rs (function return)
jalr rs, rd
Jump and Link Register
001001
Jump to rs, save address in rd
rd = PC + 4, PC = rs
