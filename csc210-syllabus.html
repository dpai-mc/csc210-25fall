CSC210: Computer Organization & Architecture

3-Credit Online Lecture Course

Course Description

This course will explore the organization and behavior of real computer systems at the assembly-language level. The mapping of statements and constructs in a high-level language onto sequences of machine instructions is studied, as well as the internal representation of simple data types and structures. Numerical computation is examined, noting the various data representation errors and potential procedural errors. (C-ID: COMP-142)

Student Learning Objectives

Upon successful completion of this course, students will be able to:

  • Write, execute and debug small programs or segments of larger programs using an assembly language.
  • Describe basic computer architecture, organization and design.
  • Discuss how fundamental high-level language constructs are implemented at the machine-language level.

17-Week Course Outline

Phase 1: Digital Foundations (Weeks 1-4)
Week 1: Introduction to Computer Architecture
Topics: Course overview, levels of abstraction, number systems
Reading: "But How Do It Know?" Ch. 1-2 + supplementary materials
Assignment: Number conversion problems, reflection on hardware-software interface
Videos: Introduction to binary/hex, computer abstraction layers
Week 2: Boolean Logic & Gates
Topics: Boolean algebra, truth tables, basic logic gates
Reading: "But How Do It Know?" Ch. 3-4
Assignment: Boolean expression simplification, gate-level design problems
Videos: Logic gate operations, Boolean algebra rules
Checkpoint 1: Simple gate combinations in Logisim (verify understanding)
Week 3: Combinational Logic Design
Topics: Adders, multiplexers, decoders, encoders
Reading: Supplementary materials on combinational circuits
Assignment: Design problems for arithmetic circuits
Videos: Building complex circuits from gates
Week 4: Sequential Logic & Memory Basics
Topics: Latches, flip-flops, registers, RAM concepts
Reading: "But How Do It Know?" Ch. 5-6
Assignment: Sequential circuit analysis, memory organization problems
Quiz 1: Covers weeks 1-4
Checkpoint 2: 8-bit register design submission
Phase 2: Assembly Language (Weeks 5-8)
Week 5: Introduction to MIPS Architecture
Topics: ISA concepts, MIPS registers, instruction formats
Reading: MIPS reference materials, instruction set overview
Assignment: Instruction encoding/decoding exercises
Videos: MIPS architecture overview, MARS tutorial
Week 6: MIPS Arithmetic & Logic Instructions
Topics: R-type instructions, immediate operations, logical operations
Reading: MIPS programming guide (arithmetic/logic sections)
Assignment: Write simple MIPS programs, trace execution
Videos: Step-by-step MIPS programming examples
Checkpoint 3: 8-bit ALU simulation due (matching "But How Do It Know?" specs)
Week 7: MIPS Control Flow & Procedures
Topics: Branches, jumps, procedure calls, stack usage
Reading: MIPS programming guide (control flow sections)
Assignment: Convert Java methods to MIPS procedures
Videos: Stack operations, procedure calling conventions
Week 8: MIPS Memory & Arrays
Topics: Load/store instructions, addressing modes, array manipulation
Reading: MIPS memory model documentation
Assignment: Array processing programs in MIPS
Midterm Exam: Comprehensive exam on weeks 1-8
Phase 3: CPU Design Theory (Weeks 9-12)
Week 9: CPU Architecture Overview
Topics: Von Neumann model, datapath components, fetch-decode-execute
Reading: "But How Do It Know?" Ch. 7-9
Assignment: Datapath tracing exercises, component identification
Videos: CPU simulation walkthroughs
Checkpoint 4: Complete RAM module with MAR (256 bytes)
Week 10: The ALU & Register File
Topics: ALU design, operation selection, register file organization
Reading: "But How Do It Know?" Ch. 10-11
Assignment: ALU operation analysis, design modifications
Project Planning: Submit CPU implementation plan
Week 11: Control Unit Design
Topics: Instruction decoding, control signals, microprogramming basics
Reading: "But How Do It Know?" Ch. 12-13
Assignment: Control signal generation for instruction subset
Videos: Control unit implementation strategies
Checkpoint 5: Register file with bus connections (R0-R3, PC, MAR, etc.)
Week 12: Memory Interface & I/O
Topics: Memory addressing, I/O mechanisms, memory-mapped I/O
Reading: "But How Do It Know?" Ch. 14-15
Assignment: Memory system design problems
Quiz 2: Covers weeks 9-12
Phase 4: Integration & Modern Topics (Weeks 13-17)
Week 13: Complete CPU Integration
Topics: Connecting all components, instruction implementation
Reading: "But How Do It Know?" Ch. 16-17
Assignment: CPU design documentation, instruction set planning
Videos: Complete CPU walkthroughs
Checkpoint 6: Basic CPU with MOV, ADD, SUB working
Week 14: Advanced Instructions & Testing
Topics: Implementing jumps, branches, and I/O instructions
Reading: Supplementary materials on instruction implementation
Assignment: Test program development
Project Progress: Instructor feedback on checkpoint 6
Week 15: Performance Basics (Shortened Week - 2 days)
Topics: CPI, clock rate, basic performance metrics
Reading: Performance measurement introduction
Light Assignment: Performance calculation worksheet
Week 16: Modern Architecture Topics
Topics: Cache basics, pipelining introduction, multicore concepts
Reading: Selected modern architecture articles
Assignment: Research paper on modern CPU feature
Checkpoint 7: Full CPU with 10+ instructions
Week 17: Course Synthesis & Final Project
Topics: Review, industry applications, future directions
Final Project Due: Complete CPU simulation with documentation
Final Exam: Comprehensive exam (proctored online)
