Carry In Circuit

We need to set the Carry In for the ALU which will come from the Carry output from the Control Section

Ctmp is a small circuit that has 2 inputs, S from Temp1 and C from the Carry Out of the ALU (comes out from the Flags link).

The output from the Ctmp circuit is ANDed with the Carry from the Control System

Tunnels S comes off the TMP_S output from the Control Section. The Tunnel C comes off the Control output, shown above.

The Ctmp circuit details include 2 inputs, S and C with a single output into the AND gate. The details of Ctmp is that it is a simple 1 bit memory circuit.
