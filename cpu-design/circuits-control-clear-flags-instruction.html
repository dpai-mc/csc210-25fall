Control Unit for Clear Flags Instruction

Learn how the Clear Flags instruction resets all ALU status flags to prepare for the next operation.

Clear Flags Instructions

The Operation instruction is an 8-bit value that is broken down as follows:

bit 0 bit 1 bit 2 bit 3 bit 4 bit 5 bit 6 bit 7
0 for Non-ALU Instr. 1 1 0 x x x x

Bit Definitions

bit 0: When this is 0, we have a non-ALU instruction (such as Load, Store, Jump IF, or Clear Flags)

bits 1-3: Are 110 for a Clear Flags instruction

bits 4-7: Are not used for this instruction (marked as x, can be any value)

Clear Flags Instruction

A Clear Flags instruction sets all of our Jump IF flags (C, A, E, Z) to 0. This resets the flag register before the next operation.

Instruction Mnemonic: CLF

Clear Flags Implementation

The implementation of Clear Flags is accomplished by setting the FLAG_S (Flags Set) signal to 1 at the appropriate time in the operation cycle. This timing is critical:

When the BUS contains all zeros (0x00) and BUS_1 is set to 1, all of the flag outputs from the ALU are False (0):

  • Carry out = 0
  • A is larger than B = 0
  • A equals B = 0
  • Result equals 0 = 0

By setting FLAG_S = 1 when the BUS is all zeros, all flags are captured as 0, effectively clearing them before the next operation begins.
