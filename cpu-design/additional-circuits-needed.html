Additional Circuits Required for the CPU

To fully complete the CPU design, several auxiliary circuits and components are required in addition to the main **RAM, ALU, and Control Unit**. The conceptual layout below highlights these components:

List of Required Auxiliary Components

  • **R0, R1, R2, R3** - General-purpose Registers on the CPU
  • **TMP** - A Temporary Register for storing data sent to ALU input B
  • **bus1** - A controller switch (Multiplexer) for ALU input B
  • **clk** - A Clock component to synchronize operations
  • **ACC** - An Accumulator Register to control and sync all outputs from the ALU
  • **PC** - Program Counter Register
  • **IR** - Instruction Register

Component Details

Registers (R0-R3)

These are **8-bit memory registers**. They should use the same core design as the memory cells created when building your RAM. We will use R0-R3 instead of R1-R4 for consistency.

TMP (Temporary Register)

This is a standard **8-bit memory circuit**. You should create a new circuit for TMP with a distinct circuit appearance (input at the top, output at the bottom, and set control on the right) for clear identification in the main CPU circuit.

bus1 Controller

This is a critical control element required for selecting the appropriate input for **ALU input B**. Follow the link for a detailed circuit explanation:

  • bus1 Circuit Details

clk (Clock)

This component is a clock simulator required for synchronized stepping through the CPU operations. Follow the link for detailed instructions on setting this component up:

  • Clock Component Setup

ACC (Accumulator)

The Accumulator is just another **8-bit Register** circuit. It is used to temporarily store the result of the ALU operation before it is written back to a general-purpose register or memory.

PC (Program Counter) and IR (Instruction Register)

Both of these are **8-bit Register** circuits and should utilize the same core memory design you created earlier. The PC holds the address of the next instruction, and the IR holds the current instruction being executed.
