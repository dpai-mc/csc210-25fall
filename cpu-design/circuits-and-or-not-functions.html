Circuits: AND, OR, NOT Functions

Implementing 8-bit Logic Functions

These circuits are essentially just extensions of the individual Gates. An **AND** gate, for instance, takes 2 bits and ANDs them together. For these functions, we need to do the same but instead of using two 1-bit inputs, we now have two **8-bit inputs** that we are AND/OR/NOT the two together.

8-bit AND Function Example

For an 8-bit **AND** function, where you have two 8-bit inputs designated as "a" and "b", you would perform the operation **bit by bit**:

  • **AND** the position **0** bits of both together and place the result on the 0 bit position output.
  • **AND** the position **1** bits of both together and place the result on the 1 bit position output.
  • etc ... for bits **2 through 7**.

OR and NOT Functions

The same **bit-by-bit process** would be done for the **OR** function.

The **NOT** function only has one 8-bit input where you **NOT** each individual bit as it passes through to the 8-bit output.
