Layout CPU

The first thing you need to do is to create an 8-bit bus. This can be done by creating concentric rectangles using the "Wiring Tool" in the "Base" folder. The wiring tool allows you to create 1/2 a rectangle at a time, so you create one half and then create the 2nd half.

I would error on making this too large as opposed to too small.

After laying out the BUS, you need to input and attach all of your components. Remember that each 8-bit input and output is going to need to be tied to the BUS using a wiring fan. I used the inner loop on the BUS as bit0 and worked my way outwards so that the outermost loop is bit7. It is important that you do this consistently for each and every component in your CPU.
