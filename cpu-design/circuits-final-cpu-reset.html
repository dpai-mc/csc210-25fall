RESET

We need to set a "RESET" input that will allow us to set all registers and memory locations to hold all 0's. At the CPU level, there are 2 places where we want to have the reset signal sent: The Control Section and to RAM.

We use a 1-bit input for RESET that goes into the Control Section. We also set a Tunnel from this input so that we can use this same input as the RESET input for RAM
