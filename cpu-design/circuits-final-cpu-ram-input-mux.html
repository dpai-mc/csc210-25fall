RAM Input MUX

We need to setup the RAM input so that we can take input off the BUS and also to read input in through an 8-bit input device. To do this, we need a multiplexor.

The basic top-level design in Logisim is as shown below:

The MUX is found in the Plexers folder and is named "Multiplexor"

The design details for the Multiplexor is to set the "Set Bits" item to "1". Also, the top input (labeled 0) should be wired to the BUS while the bottom input (Input 1) should be wired to an 8-bit input pin, in this case named "RAM INPUT".

The "LOAD" tunnel should be wired to the left of the 2 MUX connections along the bottom of the MUX. The "LOAD" tunnel is covered on Circuits: Final CPU: LOAD, MARS, END.
