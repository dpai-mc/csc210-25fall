Circuits: XOR Comparator

XOR Comparator Functionality

For this function, we need to create a circuit that performs **three different operations** simultaneously on two 8-bit inputs:

  1. Output the results from a bit-wise **Exclusive OR (XOR)** for the two 8-bit inputs. This results in an **8-bit output**.
  2. Output if the two 8-bit inputs are **equal**. This is a **1-bit output** (**1** if equal, **0** if not equal).
  3. Output if input "**a**" is **larger** than input "**b**" over all 8 bits. This is a **1-bit output** (**1** if "a" is larger, **0** if equal or less than "b").

We will work our way up to the final 8-bit version by progressing through the steps below.

Chaining for "Is Larger" and "Is Even"

To determine if 'a' is larger than 'b', we must process the bits from **most significant (Bit 7) to least significant (Bit 0)**. We use two additional inputs to chain these single-bit comparators:

  • **pl** (Prior Is Larger): Input from the next higher bit comparison.
  • **pe** (Prior Is Equal): Input from the next higher bit comparison.

For the **most significant bit (MSB)**, these inputs will be constant values: **pl** will be **0 (Ground)** and **pe** will be **1 (Constant 1)**.

The **"Is Larger"** output is calculated using the following logic gates: $$ L = (a \cdot \neg b \cdot pe) + pl $$ This means 'a is larger' if:

  • 'a' is 1, 'b' is 0, **AND** the prior bits were equal (pe=1). **OR**
  • A prior, more significant bit already determined 'a is larger' (pl=1).

The Complete 8-bit XOR Comparator

To create the full 8-bit comparator, you chain **eight** of the single-bit comparator circuits from the previous step. The connection is done from **Bit 7** (MSB) down to **Bit 0** (LSB).

  • The **'l'** (is larger) output of bit $N$ connects to the **'pl'** (prior is larger) input of bit $N-1$.
  • The **'e'** (is equal) output of bit $N$ connects to the **'pe'** (prior is equal) input of bit $N-1$.

The **final outputs** are taken from the least significant bit (Bit 0):

  • The final **"is larger"** evaluation is the **'l'** output from the Bit **0** comparison.
  • The final **"is equal"** evaluation is the **'e'** output from the Bit **0** comparison.

The bit-wise XOR result comes from the **'o'** (XOR result) output of all 8 components, forming the final 8-bit output.
