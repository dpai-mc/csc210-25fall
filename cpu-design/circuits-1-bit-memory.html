1-bit Memory

A 1-bit memory circuit is the basis for all storage in our CPU. It is often created using two cross-coupled NOR gates (or two cross-coupled NAND gates). The basic circuit is called a "Set-Reset (S-R) Latch."

Set-Reset (S-R) Latch

The S-R Latch is a fundamental sequential logic circuit. It has two inputs, Set (S) and Reset (R), and two complementary outputs, Q and Q-not. The circuit stores a single bit of information.

The standard S-R Latch truth table (using NOR gates) is:

  • S=0, R=0: Hold state (Memory)
  • S=1, R=0: Set Q to 1
  • S=0, R=1: Reset Q to 0
  • S=1, R=1: Invalid/Forbidden state

In Logisim, the S-R Latch is often simplified with a clock or enable input to control when the inputs affect the output (this is called a Gated Latch or a D-Latch, but for simplicity, we are focusing on the S-R core using the Logisim Platform for this discussion).

For my final circuit appearance for the 1-bit memory circuit, I just used a square to represent the circuit.
