FLAGS Configuration

We need to configure the flags for

  • C - Carry out
  • A - A larger
  • E - A equal B
  • Z - output is Zero

which are all outputs from the ALU (inputs to the Control Section).

We also need to connect the FLAGS_S (Flags Set) bit from the Control Section to the FLAGS circuit which will allow us to set the different Flag values for input into the Control Section.
