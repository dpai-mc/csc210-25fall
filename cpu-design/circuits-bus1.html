CIRCUITS: Bus 1

Instructional guide for building the Bus 1 circuit.

Bus 1 Circuit

This "bus1" circuit is used to put a value of 1 (0000 0001) for Input B into the accumulator whenever the "bus 1" bit is 1. When the "bus 1" bit is 0, we pass the value from TMP to Input B.

To do this, we start with an 8‑bit input coming from TMP, an 8‑bit output going to Accumulator B, and a 1‑bit input for bus 1.

Design & Logic

To make sure the output for the 0‑bit is 1 whenever the bus‑1 bit is 1—and the value stored in TMP when bus‑1 is 0—we OR the two values together and take that as bit0 out.

For bits 1–7, we want them to be 0 whenever bus 1 is 1, and equal to TMP whenever bus 1 is 0. We negate the bus 1 signal and AND it with each TMP bit. Thus when bus 1 is 1, bits 1–7 are zero; when bus 1 is 0, bits 1–7 pass through unchanged.

Testing

As always, test your circuit to ensure that when bus‑1 input is 1, output is 0000 0001, and when bus‑1 input is 0, output equals TMP. Lastly, adjust the circuit appearance to make it easier to incorporate into your final CPU.
