Important Notes on Registers

Program Counter (PC):

  • The PC is the **Program Counter** which holds the address of the next instruction to be processed.
  • This is simply an **8-bit register** circuit.

Instruction Register (IR):

  • The IR stands for **Instruction Register** which holds the current instruction being processed.
  • This is just an **8-bit memory circuit**.
