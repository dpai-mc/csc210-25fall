Clock Input Modification

The "Clock" needs to be modified so that we no longer have to manually cycle it. In order to run programs, we are going to invoke the "Ticks Enabled" option to automatically run our Assembly programs in the CPU. In order to do this, we need to change the clock to a single input and also to add the "End" bit which will disable the clock cycling at the end of our assembly program.

Since the "END" tunnel is "NOTted" and then "ANDed" with the Clock input, whenever "END" is high, the output from the AND gate is low so the clock never cycles.
