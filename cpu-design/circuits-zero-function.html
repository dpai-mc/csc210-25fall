Zero Function

The **Zero function** is a circuit that is used to identify when all **8-bits** of the output are **0**. This means that the circuit returns **1** when all of the inputs are **0** and **0** when any of the inputs are **1**.

An **8 input OR gate** would return **0** when all inputs are **0** and **1** when any of the inputs are **1**. So, a Zero function is simply a **NOTted OR Gate**.

**Remember, 8 bits in, 1 bit out for this Function.**

A simple, required component for the CPU's Arithmetic Logic Unit (ALU) flags.
