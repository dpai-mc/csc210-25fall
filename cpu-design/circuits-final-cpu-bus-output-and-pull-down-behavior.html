BUS Output and Pull Down Behavior

We need to add an 8-bit output to our "Bus" so that we can see at the next higher level what is on the "Bus" at any given point in time. Additionally, we want to give all of the connections to the "Bus" a "Pull Down" behavior meaning that every time we start up the simulation, these will all "Pull Down" to the state of "0" ensuring that our "Bus" always starts out empty.

In order to set elements to "Pull Down," you will have to edit those specific circuits that hold the input and/or output to be modified. For example, you can modify all of the registers (R0-R4, assuming you used the same circuit for each) by opening up the 8-bit register circuit and checking/changing the input and output behavior to "Pull Down."

Items that should be set to "Pull Down"

Inputs/Outputs for:

  • R0-R4
  • ALU
    • 8-bit inputs a, b
    • 8-bit output c
    • 1-bit inputs a, b, c for Operation Code
    • 1-bit outputs a larger, equal, zero, and carry out
  • TMP
    • 8-bit input and output
  • Accumulator
    • 8-bit input and output
    • 1-bit inputs for s and e
  • BUS-1
    • 8-bit input and output
    • 1-bit input "bus1"
  • PC
  • IR
  • RAM
    • 8-bit "address", i (input), o (output)
    • 1-bit reset
