Final Submission Requirements

For the Final Submission, you need to complete the following based on the desired grade:

For a Grade of "C":

C-Grade Progress: 0/2 Requirements Met

For a Grade of "B":

  • Complete "For Grade of C" **AND**
  • Complete the integrated controls section: Circuits: Final Control Unit Circuit
  • Create a "**Final**" CPU with all parts: (Circuits: Final CPU)

For a Grade of "A":

  • Complete "For Grade of B" **AND**
  • Create the **top-level CPU for simulation purposes** (Circuits: Top-Level CPU)
  • Download the provided Program and run it (5 * 5 program - MYSIM.txt)
