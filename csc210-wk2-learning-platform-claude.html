Boolean Algebra Laws Reference

Identity Laws

A · 1 = A
A + 0 = A

Null Laws

A · 0 = 0
A + 1 = 1

Complement Laws

A · A = 0
A + A = 1

Commutative Laws

A · B = B · A
A + B = B + A

Associative Laws

(A · B) · C = A · (B · C)
(A + B) + C = A + (B + C)

De Morgan's Laws

NOT(A · B) = A + B
NOT(A + B) = A · B
