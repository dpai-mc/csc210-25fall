Step 3: Implement getMin Procedure

This procedure finds the minimum of two integers using conditional branching:

# ======================================= # PROCEDURE: getMin # Purpose: Find the minimum of two integers # Parameters: $a0 = first integer # $a1 = second integer # Returns: $v0 = minimum value # ======================================= getMin: # Compare the two values blt $a0, $a1, first_smaller # Branch if $a0 < $a1 # If we reach here, $a1 <= $a0 move $v0, $a1 # Return second value (smaller) jr $ra # Return to caller first_smaller: move $v0, $a0 # Return first value (smaller) jr $ra # Return to caller

‚úÖ Analysis:

This procedure demonstrates conditional branching. Notice how we use blt (branch less than) to compare the values and move to set the return value. The procedure follows MIPS calling conventions by using $a0/$a1 for parameters and $v0 for the return value.

Step 5: Implement sumRange Procedure (Most Complex)

This procedure calculates the sum from the smaller to the larger number using a loop:

# ======================================= # PROCEDURE: sumRange # Purpose: Calculate sum from smaller to larger number # Parameters: $a0 = first integer # $a1 = second integer # Returns: $v0 = sum from min to max (inclusive) # Example: sumRange(7, 15) = 7+8+9+10+11+12+13+14+15 = 99 # ======================================= sumRange: # Save registers we'll modify addi $sp, $sp, -8 # Allocate stack space sw $s0, 4($sp) # Save $s0 sw $s1, 0($sp) # Save $s1 # Determine which is smaller and which is larger blt $a0, $a1, order_correct # Branch if $a0 < $a1 # If we reach here, need to swap values move $s0, $a1 # $s0 = smaller value move $s1, $a0 # $s1 = larger value j start_sum # Jump to sum calculation order_correct: move $s0, $a0 # $s0 = smaller value move $s1, $a1 # $s1 = larger value start_sum: li $v0, 0 # Initialize sum to 0 move $t0, $s0 # Initialize counter to smaller value sum_loop: add $v0, $v0, $t0 # Add current number to sum addi $t0, $t0, 1 # Increment counter ble $t0, $s1, sum_loop # Continue if counter <= larger value # Restore saved registers lw $s1, 0($sp) # Restore $s1 lw $s0, 4($sp) # Restore $s0 addi $sp, $sp, 8 # Deallocate stack space jr $ra # Return to caller

‚úÖ Analysis:

This procedure demonstrates several important concepts:

  • ‚Ä¢ Stack Management: Saves/restores $s0 and $s1 registers
  • ‚Ä¢ Conditional Logic: Determines correct order of values
  • ‚Ä¢ Loop Structure: Uses counter-controlled loop with ble
  • ‚Ä¢ Accumulator Pattern: Builds sum incrementally
