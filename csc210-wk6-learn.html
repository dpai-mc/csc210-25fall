Quick Reference

R-Type Instructions

add rd, rs, rt
sub rd, rs, rt
and rd, rs, rt
or rd, rs, rt
xor rd, rs, rt
nor rd, rs, rt
sll rd, rt, shamt
srl rd, rt, shamt
sra rd, rt, shamt

I-Type Instructions

addi rt, rs, imm
andi rt, rs, imm
ori rt, rs, imm
xori rt, rs, imm
slti rt, rs, imm
lui rt, imm

Common Registers

$t0-$t7: Temporary
$s0-$s7: Saved
$a0-$a3: Arguments
$v0-$v1: Return values
$zero: Always 0
$ra: Return address

Instruction Formats

R-Type Format

op(6) | rs(5) | rt(5) | rd(5) | shamt(5) | funct(6)

Used for register-to-register operations

I-Type Format

op(6) | rs(5) | rt(5) | immediate(16)

Used for immediate operations and loads/stores
